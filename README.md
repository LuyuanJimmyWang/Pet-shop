# Pet-shop
Required Software and Package with Version: \
Node.js 18.16.1 \
lite‐server 2.6.1 \
sodility 0.5.0 \
web3 4.1.0 \
truffle 5.11.2 \
Ganache 2.7.1 \
Frontend Framework: None \
Setup instuctions: \
Install Node.js and npm at https://nodejs.org/en \
Install git at https://git‐scm.com/ \
Install ganache at https://trufflesuite.com/ganache/ \
Install metamask at https://metamask.io/download/ \
Once we have those installed \
run \
npm install web3 \
npm install ‐g truffle \
npm install ‐g solc \
npm install ‐‐global lite‐server \
Start a ganache server and connect it to metamask by enter the 12 Secret Recovery Phrase in
ganache. \
Add a network in metamask of RPC server HTTP://127.0.0.1:7545 and chain ID 1337. \
Then enter the pet‐shop repository \
run \
truffle compile \
truffle migrate \
npm run dev \
